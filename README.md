# TaskFlow - Modern To-Do App 🚀

A full-stack Flutter application built for task management with a focus on UI/UX and clean architecture.

## ✨ Features
- **Auth**: Firebase Email/Password Sign-up (with 6-char validation) & Login.
- **Database**: Realtime syncing via Firebase REST API.
- **UI**: Dark/Light mode support with smooth animations.
- **State Management**: GetX for reactive updates and clean routing.

## 🛠️ Installation
- Download the APK from the [Releases](link-to-your-release) tab.
- Or run `flutter run --release`
